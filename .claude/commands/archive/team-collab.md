# Team Collaboration Commands

## Share Knowledge
/share [what] with [who]
- Package specified knowledge
- Strip personal information
- Share with team member
- Log sharing action

## Request Help
/ask-team [question]
- Post to team channel
- Include relevant context
- Aggregate responses
- Synthesize answers

## Delegate Task
/delegate [task] to [person]
- Create task with full context
- Assign to team member
- Set up check-in schedule
- Track completion

## Team Status
/team-status
Show:
- Active team members
- Current tasks by person
- Shared knowledge updates
- Recent collaborations
- Team performance metrics

## Merge Work
/merge-work from [person]
- Pull their branch/context
- Identify conflicts
- Merge changes
- Update shared memory
- Notify completion
